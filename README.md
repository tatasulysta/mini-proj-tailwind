# clickManage – Productivity Landing Page

## 🚀 Key Features

- **Multilingual Support**: Full localization in English and Indonesian with persistent language selection.
- **Phosphor Icons**: Consistent, high-quality iconography throughout the application.
- **Optimized Performance**: Built on Next.js with optimized font loading and zero-bloat asset management.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (Pages Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Phosphor Icons](https://phosphoricons.com/)
- **Localization**: [i18next](https://www.i18next.com/) with `react-i18next`
- **Fonts**: [Poppins](https://fonts.google.com/specimen/Poppins) (via `next/font`)

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- pnpm (Recommended) or npm/yarn

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd mini-proj-tailwind
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
